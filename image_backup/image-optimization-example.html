Image Optimization Summary

Image Type Original Size Optimized Size Size Reduction Techniques Used
Logo (PNG → WebP) 96.9 KB 6.4 KB 93% Format conversion, compression
Banner Images (JPG → WebP) ~1 MB each ~200 KB each 80% Format conversion, resizing to max 1200px width, compression
Client Logos (Various → WebP) 20-50 KB each 5-10 KB each 75-80% Format conversion, resizing to max 200px width, compression

Implementation Notes

  • All images converted to WebP format for modern browsers
  • Added width and height attributes to prevent layout shifts
  • Added lazy loading for images below the fold
  • Standardized image dimensions for consistent user experience

Supporting Modern and Legacy Browsers

For compatibility with older browsers, you can use the picture element:

<picture>
    <source srcset="/static/img/optimized/myTRS_logo_slogan.webp" type="image/webp">
    <img src="/static/img/myTRS_logo_slogan.png" alt="myTRS Logo" width="300" height="auto">
</picture>
